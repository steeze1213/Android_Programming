# Android Programming

학교 수업 모바일 프로그래밍 및 실험을 한 학기 동안 수강하며 실습, 과제로 작성한 코드 모음입니다.  
파이썬 기초 문법부터 데이터 처리, 시각화, 머신러닝·딥러닝 기초까지 단계적으로 학습한 개인 학습 기록용 레포지토리입니다.

---

## 사용 교재

IT CookBook, Android Studio를 활용한 안드로이드 프로그래밍
- 출판 연도: 2023

---

## 학습 내용 요약

### Chapter 01 – 안드로이드 개요 및 개발 환경
- 안드로이드 개요, 개발 환경 설치

### Chapter 02 – 첫 안드로이드 앱 만들기
- Hello Android, AVD 사용, 프로젝트 구조

### Chapter 03 – 안드로이드를 위한 Java 문법
- Java 기초, 클래스, 상속

### Chapter 04 – 기본 위젯
- TextView, Button 등 기본 UI 위젯

### Chapter 05 – 레이아웃
- LinearLayout 및 기타 레이아웃

### Chapter 06 – 고급 위젯과 뷰 컨테이너
- 고급 UI 컴포넌트, 뷰 컨테이너

### Chapter 07 – 메뉴와 대화상자
- 메뉴, 토스트, 다이얼로그

### Chapter 08 – 파일 처리
- 파일 입출력 기초 및 응용

### Chapter 09 – 그래픽과 이미지
- 그래픽 처리, 이미지 표시

### Chapter 10 – 액티비티와 인텐트
- 액티비티 전환, 인텐트 활용

### Chapter 11 – 어댑터뷰
- 리스트뷰, 그리드뷰, 스피너

### Chapter 12 – 데이터 저장과 관리
- SQLite 데이터베이스 활용

### Chapter 13 – 멀티미디어와 구글 지도
- 오디오, 스레드, 구글 지도 API

### Chapter 14 – 서비스와 브로드캐스트 리시버
- 서비스, 브로드캐스트 리시버, 콘텐츠 프로바이더

### Chapter 15 – 앱 배포
- 배포용 앱 생성, 구글 플레이 등록
